It uses a ky-037 sound detector and a SPI TFT LCD screen to display
the results from the sound detector.

# How it works

By using a ky-037 sound detector i can measure the delta(the difference
between quiet and noice) to display the result in a sound wave like pattern.
I had som trouble making the screen work because of the backlight leakage. But
with the help of some resistors i solved that problem.
